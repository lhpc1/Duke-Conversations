END POINTS

  • POST "/auth" with valid username and password credentials will return a JSON web token
  • POST "/admin/register" with "username" and "password" will create a new admin user
  • GET "/professors" will return a list of all professors
  • GET "/dinners" will return a list of all dinners
  • GET "/students" will return a list of all students
  • GET "/professor/[string:uniqueID]" will return a specific professor by Unique ID
  • GET "/student/[string:netID]" will return a specific student by netID
  • GET "/dinner/[int:id]" will return a specific dinner by id
  • POST "/professor/register" will create a new professor object in memory
    • "uniqueID": String
    • "firstName": String
    • "lastName": String
    • "email": String
    • "genderPronouns": Integer
    • "department": Integer
    • "title": Integer
    • "school": Integer
  • POST "/dinner/register" will create a new professor object in memory
    • "timeStamp": String
    • "topic": String
    • "description": String
    • "studentLimit": Int
    • "address": String
    • "dietaryRestrictions": String
    • "professorID": String
  • POST "/student/register" will create a new professor object in memory
    • "netID": String
    • "uniqueID": String
    • "firstName": String
    • "lastName": String
    • "genderPronouns": Int
